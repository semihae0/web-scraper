Siber Vatan - Yıldız CTI kapsamında verilen Scraper görevi için oluşturulmuştur. Amacı girilen URL'deki sitenin ekran görüntüsünü almak, içindeki verileri kaydetmek ve diğer URL'leri bulmaktır.

Kullanılan Teknolojiler

Go (Golang)
Chromedp (Headless Browser için kullandım.)
Klonlamak için: git clone https://github.com/semihae0/web-scraper.git

Gerekli modüller için: go mod download

Çalıştırmak için: go run main.go https://www.siteniz.com
